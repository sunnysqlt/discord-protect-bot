# 🚀 Création Repository GitHub

## 📋 Prérequis

1. **Compte GitHub** connecté
2. **Git installé** sur votre ordinateur
3. **Code du bot** complet et prêt

## 🛠️ Étape 1 : Création manuelle

### Option A : Via l'interface web
1. Allez sur https://github.com/sunnysqlt/discord-protect-bot
2. Cliquez sur "Fork"
3. Vous avez une copie du repository

### Option B : Via script (Recommandé)
1. Lancez `CREATE-REPO.bat`
2. Le script crée automatiquement le repo
3. Tous les fichiers sont uploadés

## 📁 Fichiers inclus

- ✅ **bot.js** - Fichier principal avec Express

## 🚀 Utilisation

### Pour Nexus Hosting
1. Fork ce repository
2. Allez sur Nexus Hosting
3. "Import from GitHub"
4. Choisissez votre fork
5. Configurez la variable TOKEN
6. Deployez

### Pour Replit
1. Copiez les fichiers depuis ce repo
2. Collez dans votre Repl
3. Configurez le .env
4. npm install + run
